## 🚀 Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**
```bash
git clone https://github.com/uxzwal/ukportfolio.git

```


2. **Install dependencies**
```bash
npm install

```


3. **Run the development server**
```bash
npm run dev

```


4. **Build for production**
```bash
npm run build

```
