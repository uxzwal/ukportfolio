# 🚀 My Professional Portfolio

## Overview
A custom-built DevOps portfolio showcasing scalable systems, automation workflows, and modern web engineering practices.

## Tech Stack
- Next.js
- Tailwind CSS
- Framer Motion
- GSAP

**Live Demo:** [uzwal.netlify.app](https://uzwal.netlify.app)

---

## 📂 Project Structure

```text
├── public/          # Static assets
├── src/
│   ├── components/  # Reusable UI components (Navbar, Hero, Projects, etc.)
│   ├── assets/      # Images and icons
│   ├── App.jsx      # Main application entry
│   └── main.jsx     # Vite entry point
├── tailwind.config.js
└── package.json

```

## 🚀 Getting Started

To get a local copy up and running, follow these steps:

1. **Clone the repository**
```bash
git clone https://github.com/uxzwal/ukportfolio.git

```


2. **Install dependencies**
```bash
npm install

```


3. **Run the development server**
```bash
npm run dev

```


4. **Build for production**
```bash
npm run build

```
